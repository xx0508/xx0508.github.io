2019-03-29

Go下载列表

二、系统要求

Go 二进制发行版可用于这些受支持的操作系统和体系结构。在继续之前，请确保您的系统符合这些要求。如果您的操作系统或体系结构不在列表中，您可以 从源代码安装或使用 gccgo.

操作系统 架构 笔记
FreeBSD 10.3 或更高版本 amd64,386 Debian GNU / kFreeBSD 不受支持
使用 glibc 的 Linux 2.6.23 或更高版本 amd64,386，arm，arm64，s390x，ppc64le 不支持CentOS / RHEL 5.x. 从源代码安装其他libc。
macOS 10.10 或更高版本 AMD64 使用 Xcode 附带的 clang 或 gcc 作为支持 cgo
Windows 7，Server 2008R2 或更高版本 amd64,386 使用 MinGW gcc。不需要 cygwin 或 msys。

三、安装Go开发环境

如果要安装旧的版本，则必须先删除现有版本。

  • 下载zip文件并将其解压缩到您选择的目录中。
  • 将 Go 安装目录下的 bin 目录（例如：C:/go/bin）添加进系统的 path 环境变量。
